GMAssert v1.0.0

Description

GMAssert is a library containing useful assertions for debugging and automated unit testing. You can use it to alert you to undesirable runtime conditions and pause the debugger at that point for further inspection.

Installation

Simply drag GMAssert.gml into your open project and import constants.txt as macros. If you wish to use the debugger, open __GMA_BREAKPOINT__ and place a breakpoint on the marked line.

Macro Listings

GMASSERT_MODE
Set the status of all assertions. Should be set to GMASSERT_MODE_ENABLED to enable assertions, GMASSERT_MODE_DISABLED to disable assertions, or GMASSERT_MODE_SELFTEST when run in the workbench.
GMASSERT_MODE_DISABLED
See GMASSERT_MODE.
GMASSERT_MODE_SELFTEST
See GMASSERT_MODE.
GMASSERT_MODE_ENABLED
See GMASSERT_MODE.
GMASSERT_TOLERANCE
The numeric error allowed by "equalish" assertions while comparing real numbers.
GMASSERT_TYPE_UNDEFINED
Used internally to identify the undefined type.
GMASSERT_TYPE_REAL
Used internally to identify the real number type.
GMASSERT_TYPE_STRING
Used internally to identify the string type.
GMASSERT_TYPE_ARRAY
Used internally to identify the array type.
GMASSERT_TYPE_UNKNOWN
Used internally to identify any type not specified above.
